UGame.JS

Просто еще один игровой движок на TypeScript.

Пока что просто тестовая версия.

Здесь только библиотека. 

# Планы

Игровой движок должен быть:

- легкий,
- компонентный,
- совместимый,
- интегрируемый,
- переносной.

Легкий. Мало весит, тратит мало ресурсов.

Компонентный. В основе лежат только базовые концепции. Все что нужно, может подключаться снаружи - через классы и адаптеры. От физики до сложных систем рендеринга, вплоть до 3d. Собственные разработки подключаются отдельно и их можно заменить на другие решения.

Совместимый. Должен обеспечивать удобную и простую интеграцию с популярными решениями на рынке: matter.js, pixi.js, three.js, havok (wasm).

Интегрируемый. Сам движок можно подключить в виде библиотеки в другие проекты. На нативном js/ts, vite/webpack. На популярных фреймворках react/vue/angular.

Переносной. Проекты легко можно билдить для разных систем: настольные ПК Windows/Linux/Mac OS, мобильные платформы Android/IOS, консоли PS/XBOX, технологии x86/ARM, web-версии. Также обеспечить процесс сборки для магазинов Google Play/AppStore/Steam/Яндекс Игры/VK Play.

# Установка

```
npm install ugame
```

или

```
yarn add ugame
```

Затем

```
import UGAME from 'ugame';
```
